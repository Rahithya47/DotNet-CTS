# Module 1 - HTML5 Basics

## Overview
This module introduces the fundamentals of HTML5. You will learn how to create a basic web page using common HTML elements.

## Learning Objectives

- Understand the structure of an HTML document.
- Use headings, paragraphs, images, and hyperlinks.
- Create simple forms.
- Organize content using semantic elements.

## Topics Covered

- HTML5 Document Structure
- Headings and Paragraphs
- Lists
- Images
- Hyperlinks
- Forms
- Tables
- Semantic Tags

## Prerequisites

- Basic computer knowledge
- Any web browser
- Visual Studio Code

## Outcome

After completing this module, you will be able to build a simple and structured web page using HTML5.
