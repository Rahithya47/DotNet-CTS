# Exercise 1 - Implementing Singleton Pattern

## Objective

Implement the Singleton Design Pattern to ensure that only one instance of the Logger class is created throughout the application.

## Design Pattern

Creational Design Pattern

## Files

- Logger.java
- SingletonTest.java

## Compile

```bash
javac Logger.java SingletonTest.java
```

## Run

```bash
java SingletonTest
```

## Expected Output

```
Logger instance created.
LOG: Application Started
LOG: Loading Configuration

Logger1 HashCode : 12345678
Logger2 HashCode : 12345678

Singleton Pattern Verified
```
