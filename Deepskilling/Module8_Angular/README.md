
# Module 8 - Angular

## Overview

Angular is a TypeScript-based front-end framework developed by Google for building scalable, maintainable, and enterprise-grade Single Page Applications (SPAs). It provides a complete ecosystem for developing modern web applications using components, services, dependency injection, routing, reactive programming, and powerful tooling.

This module introduces the core concepts of Angular and demonstrates how enterprise applications are structured using Angular best practices.

---

## Learning Objectives

After completing this module, you will be able to:

* Understand Angular architecture
* Create Angular applications using Angular CLI
* Develop reusable components
* Implement data binding techniques
* Use directives and pipes
* Build forms using Template-driven and Reactive Forms
* Implement routing and navigation
* Consume REST APIs using HttpClient
* Work with RxJS Observables
* Implement Dependency Injection
* Secure applications using Route Guards
* Build modular and scalable Angular applications

## Exercises

### Exercise 1 – Angular Project Setup

Create an Angular application using Angular CLI and understand the project structure.

### Exercise 2 – Components & Data Binding

Develop reusable components and implement different types of data binding.

### Exercise 3 – Services & Dependency Injection

Create services to share data between components using Dependency Injection.

### Exercise 4 – Routing & Navigation

Implement navigation between multiple pages using Angular Router.

### Exercise 5 – Forms & REST API Integration

Develop Reactive Forms and consume REST APIs using HttpClient.

## Best Practices

* Follow Angular Style Guide
* Organize code using feature modules
* Use reusable components
* Prefer Reactive Forms for complex applications
* Implement lazy loading where applicable
* Write clean and maintainable TypeScript code
