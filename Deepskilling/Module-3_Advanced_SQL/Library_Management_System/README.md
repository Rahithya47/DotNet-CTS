# Library Management System

## Module
Module-3 Advanced SQL Using SQL Server


## Description

Library Management System is a SQL Server based project that manages:

- Books
- Members
- Issue and Return Records
- Fine Management


## Technologies Used

- Microsoft SQL Server
- SQL
- Stored Procedures
- Transactions
- Joins
- Views


## Database

Database Name:

LibraryManagementDB


## Tables

1. Books

Stores book details.


2. Members

Stores library member information.


3. IssueRecords

Maintains book issue and return information.


4. Fine

Stores penalty details.


## Features

- Add books
- Manage members
- Issue books
- Return books
- Calculate fines
- Generate reports


## SQL Concepts Covered

- DDL Commands
- DML Commands
- Primary Key
- Foreign Key
- Joins
- Aggregate Functions
- Stored Procedures
- Transactions
- Error Handling
