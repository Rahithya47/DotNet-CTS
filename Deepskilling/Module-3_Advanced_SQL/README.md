# Module 3 - Advanced SQL

## Overview

This module focuses on advanced SQL programming concepts using SQL Server. It demonstrates the creation of relational databases, implementation of stored procedures, transaction management, and execution of advanced SQL queries.

---

## Learning Objectives

After completing this module, I learned to:

- Create relational database tables
- Define Primary Keys and Foreign Keys
- Insert and retrieve data
- Write complex SQL queries
- Create Stored Procedures
- Use Transactions (COMMIT & ROLLBACK)
- Improve database integrity

---
