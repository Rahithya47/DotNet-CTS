# Ecommerce Platform Search Function

## Objective

Implement Linear Search and Binary Search algorithms to search products in an e-commerce platform and compare their efficiency.

## Concepts Covered

- Arrays
- Linear Search
- Binary Search
- Time Complexity

## Files

- Product.java
- SearchService.java
- SearchTest.java

## Compile

```bash
javac *.java
```

## Run

```bash
java SearchTest
```

```

ID : 103
Name : Keyboard
Category : Electronics

Binary Search Result
Product Found:
ID : 103
Name : Keyboard
Category : Electronics
```
